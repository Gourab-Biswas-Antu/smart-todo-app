# Smart Todo App

A modern and clean Todo List application built using **Vanilla JavaScript** and **Tailwind CSS**.

This project allows users to manage daily tasks efficiently with a simple and beautiful interface.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks (All / Active / Completed)
- Clear completed tasks
- Persistent data using LocalStorage
- Responsive and modern UI

## Technologies Used

- HTML5
- Tailwind CSS
- JavaScript (Vanilla JS)
- LocalStorage API

## How It Works

Tasks are stored in the browser using LocalStorage, so even after refreshing the page, the tasks remain saved.
